Identity

Provides a customizable virtual drum set feature to existing online drum stores.

Problem

Drummers are looking for the best combination of kicks, snares, hi-hats, toms, and cymbals that go well with each other.

Solution

This feature allows the users to select the model they want for each drum piece, then creates a virtual drum set based on their choice so they can listen to how they sound all together.

Target Market

Online musical instrument shops/websites that are willing to improve their customers' experience of choice making

Features of the program

- Each drum piece is customizable using different models that the client business provides.
- Toms can be tuned just like the real acoustic drums.
- Allows to listen to pre-set or user made beats / licks with the customized drum set at various tempos.
- Allows to directly play with keyboard or mouse input.
